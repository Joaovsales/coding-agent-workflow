# Rule: Wrapping up a coding agent session

## Goal
Consistently close the agent session with relevant documentation updates.

## Process
Close the session, perform the following actions

1.  **Summarize the session:** Summarize the session and store it in an .md file in C:\Users\jvsal_svbe0ze\pdf-idea-generator-v2\docs\sessions-summary. Add relevant information to the summary without being verbose: what the user requested, feature or troubleshooting? What were the relevant files created and code changes? What bugs were identified and how were they fixed? Were tested created or executed? Make sure the document is relevant for context in the following sessions.
2.  **Document bugs:** Document any bugs identified in C:\Users\jvsal_svbe0ze\pdf-idea-generator-v2\tasks\issues\bug-fix-register.md, following the structure in the document.
3.  **Update testing documentation:** Update C:\Users\jvsal_svbe0ze\pdf-idea-generator-v2\TESTING.md accordingly. If we've added or changed relevant tests.
