# Product Guide: PDF Idea Generator v2

## 1. Overview

**PDF Idea Generator v2** is an intelligent content creation platform designed to transform PDF documents into AI-powered TikTok content ideas. It leverages semantic search, intelligent recommendations, and multi-style content generation to streamline the creative process for its users.

## 2. Target Audience

The platform is primarily built for:

*   **Social Media Managers:** Professionals handling multiple client accounts who need to efficiently generate a high volume of creative content ideas from source materials like reports, whitepapers, and case studies.
*   **Individual Content Creators:** Influencers, marketers, and solo entrepreneurs looking for inspiration and a structured way to develop unique content from various PDF documents.

## 3. Core Features & Priorities

### 3.1. Content Generation

The system generates TikTok scripts in various styles (e.g., "Digital Alchemy," "Generic TikTok"). While the existing styles are functional, **moderate importance** is placed on expanding this library to offer more creative variety in the future.

### 3.2. PDF Processing Pipeline

The core of the application is its 4-stage PDF processing pipeline (Parse → Chunk → Embed → Vectorize). The primary focus for future enhancement is on **optimizing the chunking and embedding process** to produce more accurate and contextually relevant semantic search results.

### 3.3. Smart Recommendations

The platform provides smart recommendations based on PDF content. The main priority for this feature is **improving the relevance and accuracy of these recommendations** to provide users with higher-quality content ideas.

## 4. Non-Functional Priorities

While the system is designed to be scalable and cost-effective, the highest non-functional priority for future development is **performance**. This includes minimizing PDF processing times and reducing search latency to ensure a fast and responsive user experience.
